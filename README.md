# Simplified-Twitter
This is a simplified twitter app.

## Technologies

- Frontend: React
- Backend: Spring Boot with Spring Security and JPA 2
- DataBase: MySql

## Features

- SignIn
- Login
- Close session
- Write message
- Edit message
- Delete mesage
- See messages
- Filter messages (username, text and date)
- reply to a message, composite structure
- like a message
